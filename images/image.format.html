PNG

PNG Stands for Portable Network Graphics, is a file format for image compression mostly used on the web and popular for application graphics TechTerms

JPEG

JPEG is A file format for image compression used mostly in Digital cameras and other photographic image capture devices. Wikipedia

GIF

A GIF is a computer file that is used on the Internet for sending images, especially moving images. GIF is an abbreviation for 'Graphic Interchange Format'. Collins dictionary
